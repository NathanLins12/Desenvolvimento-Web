RECEITA DE BOLO

Ingredientes

  • 2 Xícaras de açúcar
  • 3 Xícaras de de farinha de trigo
  • 3 Ovos
  • 4 Colheres de margarina

Modo De Preparo

  1. Bata as Claras em neve.
  2. Misture as gemas, a margarina, a farinha de trigo e o açúcar até obter uma massa homogênea.
  3. Adicione as claras em neve junto com o fermento
  4. leve ao forno
